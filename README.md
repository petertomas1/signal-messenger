# BetterSignal

**Frontend**: Vite + React + TS + Tauri
**Backend**: NestJS (WebSocket ready)

## Running Backend
```bash
cd backend
npm install
npm run dev
```

## Features Roadmap (Telegram-like)
- ✅ Real-time chat
- 🔄 Groups & Channels
- 🔐 E2EE
- 📢 Broadcasts
- 🤖 Bots
- 💰 Payments
- 📁 File sharing

More coming soon!
